Redesign the global Back button of the Pavithra Travels website to have a cinematic, premium, tourism-style appearance that feels modern and visually engaging.

OBJECTIVE:
Transform the standard back button into an attractive, smooth, animated navigation element that enhances user experience while maintaining professionalism.

------------------------------------------------------

1. POSITIONING

- Place the Back button below the header on all internal pages (except Home).
- Align it to the left within the content container.
- Ensure it is immediately visible without scrolling.
- Maintain consistent placement across all pages.

------------------------------------------------------

2. DESIGN STYLE (CINEMATIC LOOK)

Create a premium button style with:

- Rounded pill shape
- Subtle glassmorphism background (light blur effect)
- Semi-transparent background overlay
- Soft gradient using brand orange tones
- Subtle drop shadow for depth
- Smooth glow effect on hover
- Left arrow icon inside circular animated container

Button content example:
[ ⬅ ] Back to Previous Page

------------------------------------------------------

3. ANIMATION EFFECTS

On hover:
- Arrow icon slides slightly to the left (3–5px)
- Soft orange glow appears around the button
- Background gradient shifts smoothly
- Slight scale-up effect (1.03)

On click:
- Smooth micro-interaction ripple effect
- Quick fade transition

Page entry animation:
- Back button should fade in + slide from left (0.4s ease)

------------------------------------------------------

4. SCROLL BEHAVIOR

Optional premium effect:
- Slight parallax movement when scrolling
- Sticky behavior after scrolling 100px
- Soft shadow intensifies when sticky

------------------------------------------------------

5. MOBILE EXPERIENCE

- Larger tap area
- Clean minimal design
- Remove excessive animation for performance
- Maintain same cinematic style

------------------------------------------------------

6. NAVIGATION BEHAVIOR

- Use history-based navigation (navigate(-1))
- Preserve previous scroll position
- Include fallback logic if no history exists

------------------------------------------------------

7. VISUAL MOOD

The button should feel:

- Elegant
- Cinematic
- Smooth
- Modern travel portal style
- Premium tourism brand level

Avoid:
- Basic flat design
- Hard borders
- Harsh shadows
- Overly flashy effects

------------------------------------------------------

GOAL:

Create a cinematic, visually refined Back button that enhances brand perception and makes navigation feel smooth and luxurious, matching a premium All Over India tourism website.
